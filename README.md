# ระบบ OPD วังน้อยสัตวแพทย์

ระบบบันทึก OPD สำหรับคลินิกวังน้อยสัตวแพทย์ — สร้างจากดีไซน์ใน Claude Design (อยู่ในโฟลเดอร์ `_design/`)

## วิธีใช้งาน

**ดับเบิลคลิกไฟล์ `OPD วังน้อยสัตวแพทย์.html`** เปิดด้วย Chrome หรือ Edge ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม ไม่ต้องต่ออินเทอร์เน็ต (ฟอนต์สวยขึ้นเล็กน้อยถ้ามีเน็ต)

ข้อมูลทั้งหมด (เคส, คิว, สต็อก, ใบเสร็จ, นัด) **บันทึกอยู่ในเบราว์เซอร์เครื่องนั้น** (localStorage) — รีเฟรชหรือปิดเปิดใหม่ข้อมูลไม่หาย แต่ถ้าล้างข้อมูลเบราว์เซอร์ (Clear browsing data) ข้อมูลจะหายทั้งหมด ควรใช้เครื่องเดียว/เบราว์เซอร์เดียวเป็นหลัก

ระบบเริ่มต้นมาพร้อมข้อมูลตัวอย่าง (สัตว์ 6 ตัว + คิววันนี้ + สต็อก) เพื่อให้ลองกดเล่นได้ทันที

## ฟีเจอร์หลัก

- **หน้า OPD** — คิว 4 โซน (รอตรวจ → กำลังตรวจ → รอชำระ → เสร็จ), ค้นหาเคสเก่า, รับเคสใหม่/Walk-in (HN รันอัตโนมัติตามปี พ.ศ. เช่น 6900001), แถบนัดวันนี้ + สัตว์ที่แอดมิด
- **บันทึกเคส** — CC/PE/Dx/Plan, คีย์ค่าใช้จ่าย (ตัดสต็อกอัตโนมัติเมื่อรับชำระ), ปริ้นฉลากยา 🏷, อัปโหลดรูป (บันทึกลงประวัติ), แก้ไขข้อมูลสัตว์/เจ้าของ/ประวัติเก่า, ประวัติวัคซีน, นัดครั้งถัดไป
- **แอดมิด** — กดเข้าแอดมิดจากหน้าเคส คีย์การรักษาทุกวัน แล้วคิดบิลรวมทีเดียวตอนจำหน่าย
- **ใบเสร็จ/ใบกำกับภาษี** — เลขที่รันตามปี (RCP-2026-00001), VAT 3 โหมด (ไม่มี / รวมแล้ว / บวกเพิ่ม), เพ็ทช้อปไม่คิด VAT อัตโนมัติ
- **เพ็ทช้อป (POS) + สต็อก** — ขายหน้าร้าน ตัดสต็อก เตือนใกล้หมด ต้นทุน/กำไร%
- **สรุปรายรับ** — KPI, กราฟแท่ง/วงกลม, Export Excel (.csv), Export ใบเสร็จ PDF แยกเดือน
- **เอกสารภาษี** — ภาพรวม, รายงานภาษีขาย/ซื้อ, ภ.พ.30, หัก ณ ที่จ่าย (ภ.ง.ด.1/3/53 + พิมพ์หนังสือรับรอง), สรุปรายปี, เตรียมยื่น e-Filing

## โครงสร้างไฟล์ (สำหรับนักพัฒนา)

| ไฟล์/โฟลเดอร์ | คืออะไร |
|---|---|
| `OPD วังน้อยสัตวแพทย์.html` | **ตัวแอปจริง** ไฟล์เดียวจบ (รวม React, CSS, โค้ด, โลโก้) |
| `src/` | ซอร์สโค้ดแยกไฟล์ (แก้ที่นี่แล้ว build ใหม่) |
| `vendor/` | React + Babel ที่ฝังเข้าไฟล์ HTML |
| `build.sh` | สคริปต์ประกอบไฟล์ HTML — รัน `bash build.sh` (ใช้ Git Bash) |
| `serve.ps1` + `.claude/launch.json` | เซิร์ฟเวอร์ทดสอบชั่วคราว (PowerShell) |
| `_design/` | ดีไซน์ต้นฉบับจาก Claude Design (chat + prototype + รูปอ้างอิง) |

หลังแก้ไฟล์ใน `src/` ให้รัน `bash build.sh` เพื่อสร้าง `OPD วังน้อยสัตวแพทย์.html` ใหม่
